# Автоматическая оценка ответов студентов

Система автоматической оценки устных/письменных ответов студентов по шкале **2 / 3 / 4 / 5** на основе сравнения с эталонным ответом преподавателя.

## Как это работает

Оценка считается по формуле:

```
score = S^w1 × C^w2 × H^w3
grade = 2 / 3 / 4 / 5  по порогам score
```

Три компонента:

| Компонент | Что измеряет | Метод |
|-----------|-------------|-------|
| **S** | Семантическая близость ответа студента и эталона | Siamese RuBERT (дообученный) |
| **C** | Покрытие ключевых терминов из эталона | IDF-взвешенные леммы (pymorphy3) |
| **H** | Связность речи | Штраф за частицы/междометия и повторы (POS-теги) |

Оптимальные веса найдены через grid search: **S^0.78 × C^0.2 × H^0.02**

## Результаты

| Метрика | Значение |
|---------|----------|
| Accuracy (точное совпадение) | 87.36%   |
| Accuracy (±1 балл) | 98.4%    |
| MAE по оценкам | 0.174    |
| Pearson(S, оценка) | 0.915    |

Датасет: 8142 пары (вопрос, эталон, ответ студента), тематика Big Data и ML.

## Быстрый старт

**1. Построить IDF по эталонным ответам (один раз):**

```bash
python build_idf.py
```

**2. Оценить ответ студента:**

```python
from scorer import StudentAnswerScorer

scorer = StudentAnswerScorer(
    model_dir="../ass_model/models/siamese_rubert_s_128",
    weights={"w1": 0.78, "w2": 0.2, "w3": 0.02},
)

result = scorer.score(
    question="Что такое большие данные?",
    reference="Большие данные — это наборы данных большого объёма и высокой скорости накопления, для обработки которых недостаточно традиционных методов. Характеризуются моделью 4V: Volume, Velocity, Variety, Veracity.",
    student="Большие данные это огромные объёмы информации которые сложно обработать стандартными инструментами.",
    verbose=True,
)

print(result)
# {'S': 0.81, 'C_raw': 0.43, 'C': 0.62, 'H': 0.94, 'score': 0.74, 'grade': 4}
```

**3. Оценить сразу несколько ответов:**

```python
rows = [
    {
        "question":  "Что такое большие данные?",
        "reference": "Эталонный ответ...",
        "student":   "Ответ студента 1...",
    },
    {
        "question":  "Что такое MapReduce?",
        "reference": "Эталонный ответ...",
        "student":   "Ответ студента 2...",
    },
]

results = scorer.score_batch(rows)
for r in results:
    print(r["grade"], r["score"])
```

## Структура проекта

```
curs/
├── datasets/
│   ├── train.csv          # обучающая выборка (6531 примеров)
│   └── test.csv           # тестовая выборка (1611 примеров)
├── models/
│   └── siamese_rubert_s_128/  # обученная модель (скачать отдельно)
├── scorer.py              # основной модуль — S, C, H и итоговая оценка
├── train_siamese.py       # обучение Siamese RuBERT
├── build_idf.py           # построение IDF по датасету
├── evaluate.py            # оценка качества + grid search по весам
├── idf.json               # предподсчитанные IDF веса
└── README.md
```

## Формат датасета

CSV файл с колонками:

| Колонка | Описание |
|---------|---------|
| `Вопрос` | Текст вопроса |
| `Эталонный ответ преподавателя` | Правильный ответ |
| `Ответ студента` | Ответ который оценивается |
| `Оценка` | Оценка преподавателя: 2, 3, 4 или 5 |

## Переобучение на своих данных

Если хотите обучить модель на своём датасете:

```bash
# Положите свои данные в datasets/train.csv и datasets/test.csv
python train_siamese.py

# Постройте IDF по новым данным
python build_idf.py

# Найдите лучшие веса для своих данных
python evaluate.py
```
